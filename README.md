# Інтернет-магазин музичних інструментів

Це приклад MVC-проєкту для системи адміністрування інтернет-магазину музичних інструментів.

## Опис
- MVC архітектура на PHP
- модулі: Каталог продукції, Новини, Користувачі, Фотогалерея, Сторінки
- адмінська частина з додаванням, редагуванням і видаленням записів
- асинхронний обмін даними через JSON
- дані зберігаються у SQLite базі `db/database.sqlite` через PDO

## Запуск на localhost
1. Відкрийте термінал у папці `c:\Users\Admin\Desktop\coursework`
2. Запустіть PHP вбудований сервер:

```powershell
php -S localhost:8000
```

3. Відкрийте у браузері:

```
http://localhost:8000/index.php
```

## Вхід до системи
- адміністратор: `admin` / `music123`
- користувач: `user` / `user123`

## Як працює
- `index.php` маршрутизує запити через `route`
- адміністративні дії виконуються з контролера `AjaxController`
- дані зберігаються у SQLite файлі `db/database.sqlite`
- структура папок:
  - `app/Controllers` — контролери
  - `app/Models` — моделі
  - `app/Views` — представлення
  - `app/Core` — базові класи MVC
